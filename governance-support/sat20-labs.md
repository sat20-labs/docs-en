# SAT20 Labs

SAT20 Labs is the development team for SAT20 and SatoshiNet core protocols and open-source infrastructure. The team develops and maintains Indexer, STP, SatoshiNet, wallets, contracts, DEX, DAO, Explorer, and developer tools, and works with BTC communities, node operators, developers, wallets, exchanges, and AI Agent teams to build the open ecosystem.

## Current Responsibilities

1. Maintain core open-source code and documentation.
2. Advance testnet and protocol validation.
3. Support communities and developers in early integration.
4. Provide reference implementations, deployment support, and technical review.
5. Publish risks, limitations, and protocol questions still under design.
