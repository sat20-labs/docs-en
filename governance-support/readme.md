# Governance and Support

SatoshiNet is currently built by SAT20 Labs as the core protocol development team. The long-term direction is a more open structure for protocol maintenance, ecosystem support, public funding, and governance.

This section clarifies:

1. What SAT20 Labs is responsible for today.
2. The plan and boundary of a future foundation.
3. The direction of the SIP improvement process.
4. How to support protocol development and public infrastructure.
5. How to report security issues.

Supporting protocol development and GAS network economics are separate matters. Donations, sponsorships, service contracts, and future foundation arrangements do not equal buying GAS or receiving yield promises.
