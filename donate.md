# Support the SatoshiNet Ecosystem

SAT20 / SatoshiNet is long-term open-source infrastructure. You can support it through donations, code contributions, documentation, testnet validation, node operation, community cooperation, and ecosystem projects.

Supporting protocol development does not equal buying GAS, and does not represent investment, equity, yield, governance rights, or network profit promises. GAS network economics, node staking, and future foundation design are separate protocol and governance topics that are still being made public over time.

## Transparency Principles

1. Donations support SAT20 / SatoshiNet public-good development.
2. Donations do not represent investment, equity, yield, or governance promises.
3. Ecosystem cooperation should be based on real products, testnet results, public evidence, and long-term building.
4. For key public support, we will try to provide verifiable results through docs, code, transactions, demos, or project showcase pages.
