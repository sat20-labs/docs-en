# EVM Developer Preview

This page is the preview entry for Solidity / EVM developers. After EVM Runtime, RPC, Chain ID, Faucet, sample repository, and Explorer verification are stable, this page will become an executable quickstart.

## Preview Questions

1. Current supported scope of EVM Runtime.
2. How SatoshiNet's UTXO-native asset model combines with EVM semantics.
3. Boundaries of RPC, ABI, Event, Result TX, and contract state.
4. When toolchain, sample repository, and testnet entry become publicly executable.

## Required Before Quickstart

| Item | Content |
| --- | --- |
| Environment | Node.js, Foundry / Hardhat, wallet, test GAS |
| Supported versions | EVM Runtime, RPC, Solidity version |
| Network params | RPC, Chain ID, Explorer, Faucet |
| Example repo | Minimal Solidity contract and deployment script |
| Commands | install, compile, deploy, invoke |
| Expected output | contract address, txid, Result TX, state change |
| Verification | Explorer, L2 Indexer, contract state |
| Common errors | GAS insufficient, RPC mismatch, wrong contract address, Result TX missing |
