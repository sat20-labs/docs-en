# SatoshiNet AI Agent Quickstart

This page is for AI Agent developers. The goal is to let Agents read SatoshiNet docs, call wallet adapters, verify asset safety, and eventually extend toward Community Builder Agent.

## Two Paths

| Path | Goal |
| --- | --- |
| Agent Wallet & Safety | Help users safely operate wallets, STP channels, and cross-layer assets |
| Community Builder Agent | Help BTC communities design and deploy DEX, DAO, Indexer, Explorer, wallet, and contract modules |
