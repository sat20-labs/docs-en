# Community DEX / DAO Quickstart

This page guides communities in building a testnet DEX / DAO prototype. It currently defines the standard flow; deployment scripts, config files, and screenshots will be added later.

## Goal

A BTC community should be able to:

1. Choose community assets.
2. Deploy or configure a DEX frontend.
3. Configure AMM or limit order modules.
4. Configure a minimal DAO or community fund workflow.
5. Integrate wallet, Indexer, and Explorer.
6. Complete a test transaction.
7. Generate user docs and risk notes.
