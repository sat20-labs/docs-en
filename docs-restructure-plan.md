# SAT20 Docs V3 Restructure Plan

This document plans the V3 content structure for `docs.sat20.org`. It is not a protocol whitepaper; it is the product design document for Docs itself.

## New Positioning

`docs.sat20.org` is the authoritative source for SatoshiNet protocol facts, developer integration, safety verification, and implementation details. The website explains outcomes, opportunities, and action paths. Docs provide implementation, evidence, and risk boundaries.

Docs V3 goal:

> See the opportunity → choose a role → verify current capability → complete the first operation → run your own infrastructure → apply for cooperation → become a long-term builder.

## Key Changes in V3

1. Position SatoshiNet as an open Bitcoin-native execution network.
2. Add Run the Network for Mining Node, Core Node, Indexer, Explorer, RPC, monitoring, and decentralization roadmap.
3. Add Network Economics for Gas Unit, GAS asset, fee flow, node staking, distribution draft, and open questions.
4. Add Governance and Support for SAT20 Labs, future foundation, SIP, protocol support, and security disclosure.
5. Keep placeholder pages but mark status clearly.
6. Change SatoshiNet Today to three dimensions: implementation status, available environment, docs evidence.
7. Simplify roadmap to Now / Next / Later.
