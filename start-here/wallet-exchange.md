# I Am a Wallet or Exchange

Wallets and exchanges are important entry points for SatoshiNet. Users ultimately need to identify assets, deposit, withdraw, move cross-layer, trade, and verify safety through wallets and exchanges.

## What You Need to Integrate

| Capability | Role |
| --- | --- |
| L1 Indexer | Query Bitcoin L1 UTXO, assets, confirmations, and spent state |
| L2 Indexer | Query SatoshiNet UTXO, transactions, contracts, channels, and cross-layer state |
| STP State | Understand channels, splicing, unlock, lock, close, and pending transactions |
| Wallet Authorization | Keep signing, seed phrases, and passwords inside the wallet safety boundary |
| Explorer Links | Provide verifiable evidence for users and support teams |
