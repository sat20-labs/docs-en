# I Run Infrastructure

SatoshiNet needs infrastructure teams to run Core Nodes, indexers, explorers, RPC, monitoring, and data services.

Infrastructure is not peripheral. Indexers and nodes determine whether users can verify asset facts, transaction state, contract state, and cross-layer evidence.

## Participation Areas

| Area | Value |
| --- | --- |
| Core Node | Provides STP service, SatoshiNet node capability, and network base service |
| L1 Indexer | Parses Ordinals, Runes, BRC20, ORDX, and other Bitcoin L1 asset facts |
| L2 Indexer | Indexes SatoshiNet UTXO, ascend, descend, channels, contracts, and transaction state |
| Explorer | Provides visual verification for users, communities, and exchanges |
| RPC / API | Provides stable access for wallets, DEX, Agents, exchanges, and developers |
| Monitoring and Alerts | Keeps nodes, indexers, STP, contracts, and APIs running reliably |
