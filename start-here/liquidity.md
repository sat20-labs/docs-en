# I Want to Provide Liquidity

Market makers and liquidity partners can help SatoshiNet assets form real trading depth.

Liquidity participation is not only capital provision. It also includes asset pricing, routing, AMM pools, limit order markets, cross-community coordination, and risk control.

## Participation Areas

| Area | Description |
| --- | --- |
| AMM pools | Provide automated market making for community assets and base assets |
| Limit order markets | Provide order-book trading for professional users and strategies |
| Launchpad | Help new assets start distribution and initial liquidity |
| Cross-community liquidity | Connect trading and funds across BTC asset communities |
| Risk tools | Monitor price, depth, slippage, abnormal trading, and contract state |
