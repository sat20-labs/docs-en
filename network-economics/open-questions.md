# Open Design Questions

This page records GAS, node, and network-economics questions that are not finalized, so design-in-progress content is not mistaken for final protocol rules.

## Current Open Questions

1. GAS protocol name and asset identifier.
2. Differentiated staking requirements for Mining Nodes and Core Nodes, where Core Nodes include Mining Node capabilities and take additional STP responsibilities.
3. Unstaking time.
4. Malicious behavior and offline penalties.
5. Whether Core Nodes have a separate STP service fee.
6. Network security reserve release curve.
7. How node rewards adjust as real fees grow.
8. Paymaster and community gas sponsorship mechanisms.
9. GAS exit and supply reconciliation between L1 and SatoshiNet.
10. Foundation governance, Treasury Policy, transparent addresses, and legal structure.

**Page Status: Design in Progress**
