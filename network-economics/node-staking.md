# Mining / Core Node Staking

Mining Nodes and Core Nodes are planned to use GAS as staking collateral. Staking increases the cost of taking network responsibilities and provides a basis for penalty and exit rules.

## Parameters Under Design

1. GAS protocol name and asset identifier.
2. Mining Node staking threshold.
3. Core Node staking threshold.
4. Unstaking time.
5. Malicious behavior and offline penalties.
6. Whether Core Nodes have a separate STP service fee.
7. Node admission and exit flow.
