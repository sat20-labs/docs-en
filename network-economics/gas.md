# GAS: Network Fee and Security Asset

GAS is SatoshiNet's network fee and security staking asset. It connects network usage, node service, and long-term security.

## What GAS Is Used For

1. Paying transaction fees.
2. Paying smart contract execution fees.
3. Pricing network resources.
4. Preventing abuse.
5. Supporting the security staking design for Mining Nodes and Core Nodes.

## What GAS Is Not

1. GAS does not represent equity in SAT20 Labs or a future foundation.
2. GAS does not represent profit distribution, buyback, or fixed-yield promises.
3. Node staking is not a fixed-yield product.
4. The website and Docs do not provide price forecasts, yield rates, or exchange listing promises.

## Current Status

The complete GAS issuance, transcending, staking, penalties, node admission, initial distribution, and foundation governance rules still require technical simulation, public discussion, and legal review.
