# SAT20 Agent Wallet One-Minute Demo Video

This is the public video entry for the SAT20 Agent Wallet / STP drill.

The video uses real BTC testnet4 and SatoshiNet testnet evidence. It shows:

1. The wallet keeps mnemonic and signing authority under user control.
2. The STP channel places assets inside a verifiable 2-of-2 safety boundary.
3. After unlock advances the commit height, the old commitment becomes punishable.
4. Once the old commitment appears on BTC L1, the user wallet can broadcast punish and sweep funds back under user control.
