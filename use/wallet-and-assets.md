# Wallet and Assets: First Entry into SatoshiNet

This guide explains how ordinary users start from a wallet and enter SatoshiNet.

Screenshots and detailed steps will be added later.

## Preparation

1. Install [SAT20 PWA Wallet](https://sat20.org/pwa/?install=1).
2. Create or import a wallet.
3. Back up the seed phrase and unlock the wallet.
4. Confirm whether the current network is mainnet or testnet.
5. Prepare Bitcoin L1 assets or test assets.

## Verification

1. Check the txid shown by the wallet.
2. Verify L1 transactions in a Bitcoin L1 explorer.
3. Verify L2 transactions in a SatoshiNet explorer.
4. Query Indexer to verify asset location, UTXO, or channel state.
5. If STP is involved, confirm the wallet can provide a safety snapshot.
