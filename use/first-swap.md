# Complete Your First Swap

This page will document a complete user manual for completing the first swap in a SatoshiNet DEX.

## Preparation

1. SAT20 PWA Wallet is installed and unlocked.
2. Wallet is connected to the correct network.
3. Test assets and test GAS are available.
4. Target DEX domain and contract information are verified.

## Verification

1. Record the transaction txid.
2. Query transaction status in SatoshiNet Explorer.
3. Query input and output asset changes in Indexer.
4. If a contract is involved, check Result TX or contract state.
