# About SAT20 Labs

SAT20 Labs is the development team for SatoshiNet core protocols and open-source infrastructure. We develop and maintain Indexer, STP, SatoshiNet, wallets, smart contracts, DEX, DAO, Explorer, and developer tools, and work with BTC communities, developers, node operators, wallets, exchanges, market makers, and AI Agent teams to build the open ecosystem.

## Current Focus

1. Maintain unified indexer capabilities for multi-protocol Bitcoin L1 assets.
2. Advance STP channel protocol, cross-layer asset movement, and user exit safety.
3. Build SatoshiNet nodes, contract execution environment, GAS fee model, and L2 indexer.
4. Provide Wallet SDK, PWA Wallet, Agent adapter, and developer tools.
5. Help BTC communities build DEX, DAO, Explorer, wallet entry, and community infrastructure on testnet and later mainnet.

More current team responsibilities are described in [SAT20 Labs](governance-support/sat20-labs.md).

## Future Foundation

SatoshiNet plans to establish an independent non-profit foundation in the future for public protocols, open-source funding, security audits, ecosystem grants, and transparent governance. The foundation has not been established yet. Formal structure, directors, funds, and governance rules will be published after legal and governance design is completed.

See [Future Foundation](governance-support/future-foundation.md).

## Security

If you discover security issues in STP, wallets, contracts, indexers, nodes, or docs, see [Security Disclosure](governance-support/security-disclosure.md). Before a formal channel is established, contact SAT20 Labs through GitHub or official public channels and provide reproduction steps, transactions, logs, versions, and impact scope when possible.
