# Node Staking and Exit

Mining Nodes and Core Nodes are planned to use GAS as a staking asset. Full staking rules, unstaking time, penalty parameters, and admission flow remain under design.

## Current Expression Boundary

1. GAS is a network fee and security staking asset.
2. Nodes earn protocol fees according to the network services they actually provide.
3. No fixed yield or fixed APY is promised.
4. Staking, penalties, exit, and node admission parameters are subject to future formal protocol and on-chain rules.
