# Decentralization Roadmap

SatoshiNet's long-term goal is not to make all communities and applications permanently depend on SAT20 Labs. Third parties should be able to independently run nodes, indexers, explorers, wallets, DEXs, and DAOs.

## Roadmap Questions

1. Which components are currently run by SAT20 Labs.
2. What third parties can run today.
3. How node admission will open over time.
4. How indexers can be independently deployed and cross-verified.
5. What users and communities can still do if SAT20 Labs stops operating.
6. Which protocols, tools, and evidence need to be made public first.
